# 微信小程序实现计算器
### 项目说明：
微信小程序：实现一个计算器  

[数据存放：将计算后的数据存放在本地缓存中]

### 目录结构：
- pages — 存放项目页面相关文件 
	> cal 目录，存放计算器主界面，用于计算以及逻辑处理。

	> list 目录，存放历史计算之后的记录
- Screenshots — 存放项目效果图
- app.wxss — 存放项目公共样式，其他样式均存放在page目录里面
- utils — 存放utils文件，可require引入
- [临时图片](http://# "注：项目中所有图片均已转成base64")已删除
