# 微信小游戏--植树节小游戏
### 项目说明：
微信小程序：实现一个植树浇水的小游戏  

### 目录结构：
- pages — 存放项目页面相关文件
-  Screenshots — 存放项目效果图
- static下面img — 存放项目[临时图片](http://# "注：项目中所有图片均已转成base64")
- app.wxss — 存放项目公共样式，其他样式均存放在page目录里面
- utils — 存放utils文件，可require引入
